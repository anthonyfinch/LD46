# LD 46

## Theme: Keep It Alive

## Game: Shakey Hands

Entry for Ludum Dare 46. You are a cash strapped tattoo artist with debts to pay. Get tattoing and if you can do enough designs well enough, you might just make it through...


Made with:

- Godot
- Emacs
- GIMP
- Inkscape
- Asesprite
